# AI-Integration-Mini-Lab-Ai-Bot
## Project Overview

This project is an AI-powered document assistant built using
Streamlit and the Google Gemini API. Users can upload a PDF or
text document and ask questions about its content.

The AI model analyzes the uploaded document and generates
responses based only on the information contained within it.
This allows users to quickly extract key insights or clarify
information from large documents.
